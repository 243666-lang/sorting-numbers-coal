# Number Sorting Program – COAL

## 📌 Overview
This program is written in **Assembly Language** for my COAL (Computer Organization & Assembly Language) course in the 3rd semester. It takes a set of numbers from the user and sorts them in **ascending/descending** order.

## 🛠️ Technologies Used
- Language: Assembly (x86 / 8086 / MASM / EMU8086 — pick one)
- Tools: [DOSBox / EMU8086 / Visual Studio / MASM]
- Platform: Windows / Linux (with DOSBox)

## 🔧 Features
- Takes [X] numbers as input (or user decides count)
- Implements [Bubble Sort / Selection Sort / Insertion Sort]
- Displays sorted numbers
- Handles basic input validation (optional)

## 💻 How to Run

### Prerequisites
- [EMU8086] or [DOSBox + MASM/TASM]
- Basic knowledge of running Assembly programs

### Steps
1. Open EMU8086 or DOSBox
2. Load the file `sort.asm`
3. Assemble the code
4. Run the executable
5. Enter numbers when prompted
6. View the sorted output
